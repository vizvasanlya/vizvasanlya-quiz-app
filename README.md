Run quizzes, track attempts, identify weak topics, and guide learners through review paths.

## Features

- **Quiz Creation** — Build quizzes with multiple question types
- **Attempt Tracking** — Record and analyze quiz performance
- **Topic Analysis** — Identify strengths and weak areas
- **Review Paths** — Generate focused review recommendations
- **Score History** — Track progress over time
- **Search & Filter** — Find quizzes by topic, difficulty, or status
- **Analytics View** — Performance distribution and topic insights
- **Import / Export** — JSON-based quiz data portability

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (Custom Properties, Grid, Flexbox) |
| Logic | Vanilla JavaScript (ES2022) |
| Storage | LocalStorage |
| Server | Python HTTP server (dev) |

## Getting Started

### Prerequisites

- Python 3.x (for dev server)
- Modern web browser

### Installation

```bash
git clone https://github.com/vizvasanlya/First.git
cd First/vizvasanlya-quiz-app
```

### Run

```bash
npm start
```

Open [http://localhost:5173](http://localhost:5173)

### Docker

```bash
docker compose up --build
```
